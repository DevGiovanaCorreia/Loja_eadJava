## Objetivo do Software
O sistema tem como objetivo gerenciar uma loja EAD, permitindo o cadastro e controle de produtos, clientes, fornecedores, categorias e funcionários. 
Também permite registrar vendas realizadas pelos funcionários e gerar relatórios de vendas diárias.

## Funcionalidades do Sistema

### Cadastros
- Cadastro de fornecedores
- Cadastro de categorias
- Cadastro de produtos
- Cadastro de clientes
- Cadastro de funcionários

### Vendas
- Registro de vendas realizadas por funcionários
- Associação da venda ao cliente
- Registro dos produtos vendidos

### Relatórios
- Geração de relatório de vendas do dia
- Visualização das vendas realizadas pelo funcionario

## Estrutura do Sistema
O sistema é composto pelas seguintes classes principais:

- Fornecedor
- Categoria
- Produto
- Cliente
- Funcionario
- Venda
-  ItemVenda
- RelatorioDoDia

Cada classe é responsável por representar e gerenciar as informações do sistema.
